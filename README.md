### Resolucion de la practica 2 de Python


Como crear un **venv** (*virtual enviroment*), como descargar los **requirements** y como ejecutar el proyecto.
 

### Requisitos:
 - Python 3.13.12
 - jupyterlab==4.5.6
 - Virtual enviromnent
 
## Distribucion
Practica2_python:
 - notebooks   #Con notebooks para la ejecucion de ejercicios.
 - src   #Con los funcionamientos para cada ejercicio.


## Instalación
1. Creá el entorno virtual:
python -m venv .venv

2. Activalo:
   - Windows: `venv\Scripts\activate.bat`  en cmd

3. Instalá las dependencias:
pip install jupyterlab

## Cómo ejecutar

1. Iniciá JupyterLab:
poner en la consola:
`jupyter lab`

2. Navegá a la carpeta `notebooks/`
3. Abrí el notebook del ejercicio que querés ejecutar
4. Ejecutá las celdas con Shift+Enter o COntrol+Enter
